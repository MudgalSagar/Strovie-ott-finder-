# 🎬 Strovie – Movie & Series Explorer

**Strovie** is a modern web application that allows users to search, explore, and compare movies and web series. It provides detailed information, trailers, OTT availability, and even generates book-style content from movie plots.

---

## 🚀 Features

### 🔍 Search Movies & Series
Get detailed information like title, rating, genre, cast, and plot.

### 🎥 Watch Trailers
Integrated YouTube trailers for quick preview.

### 📺 OTT Platform Availability
Find where a movie or series is streaming.

### ⚔️ Compare Movies
Compare two movies side-by-side based on ratings and details.

### 📚 AI Book Generator
Convert movie plots into a book-style narrative.

### 🎠 Genre-Based Carousel
Browse movies by categories like Action, Drama, Comedy, Thriller.

### 🎨 Modern UI/UX
Dark cinematic theme with smooth animations and responsive design.

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- CSS (Custom styling)  
- React Slick (Carousel)  

### Backend
- Node.js  
- Express.js  

### APIs Used
- OMDb API (Movie details)  
- TMDb API (Additional metadata)  
- Watchmode API (OTT availability)  
- OpenAI API (Book generation)
